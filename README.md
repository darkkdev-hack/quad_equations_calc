# Quadratic Equation Solver
Current version: v0.4-beta

> **Note:** this program is optimized for desktop use. The ".exe" version will be available in the future updates.

> **Remember:** everything is subject to change.

## Requirements
1. Python;
2. No additional libraries required.

## How to use?
1. Install python on your system from the official website;
2. Download the main.py file from the Releases section;
3. Open it in any code editor (VS Code, PyCharm, IDLE, etc.);
4. Run the script (usually by pressing "F5" or "Run" button);
5. After one session, you need to press "Run" again to start the program.
