# Quadratic Equation Solver
Current version: v1.0.0

> **Remember:** everything is subject to change.

## Requirements
1. Python (you can install the latest version from the official website);
2. No additional libraries required (tkinter is built-in).

## How to use?
### Main way:
1. Download the ".exe" file from the release page of a version (e.g. v1.0 or higher *stable* versions);
2. Run it;
3. If you are done using it, just close the program as usual.


### To test the code:
1. Download the ".py" file from the release page of a version (e.g. v0.1-beta or *each* higher version);
2. Open it in any code editor (VS Code, PyCharm, IDLE, etc.);
3. Run the script (usually by pressing "F5" or "Run" button);
4. After one session, you need to press "Run" again to start the program.
